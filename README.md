# AI Hospital Disease Prediction System

A comprehensive hospital management system with AI-powered disease prediction capabilities.

## Features

### Authentication
- Admin login
- Doctor login
- Patient login
- User registration

### Patient Management
- Add new patients
- View patient records
- Update patient information
- Delete patients
- Search patients

### Doctor Management
- Add new doctors
- View doctor records
- Update doctor information
- Delete doctors
- Search doctors

### AI Disease Prediction
- Machine learning-based disease prediction
- Support for multiple symptoms
- Confidence scoring
- Risk level assessment
- Automated recommendations

### Reports
- Prediction history
- Doctor statistics
- Patient reports
- System-wide statistics
